# Radarr 手动下载电影完整教程

这篇教程教你如何通过 Radarr 的网页界面，手动搜索、选择画质、下载电影到 Samba 共享的 `/share/Movies` 目录。

---

## 一、打开 Radarr

在浏览器地址栏输入：

```
http://192.168.1.7:7878
```

> 注意：这是你家局域网地址。如果你在手机上访问，需要连家里的 Wi-Fi。

---

## 二、添加一部新电影

### 1. 点击 "Add New"

在 Radarr 首页右上角，点击 **蓝色的 "Add New" 按钮**。

### 2. 搜索电影

在搜索框输入**电影的英文名**，比如：

- `The Fast and the Furious`
- `The Matrix`
- `Inception`

> 中文名也可以试试，但英文名结果更准。

输入后会自动出现电影列表，点击你要的那一部。

### 3. 选择画质配置（Quality Profile）

点击电影后，会出现配置界面。找到 **Quality Profile** 下拉框：

| 配置名 | 含义 | 适合场景 |
|--------|------|----------|
| **Ultra-HD** | 优先下载 4K | 想要最高画质、磁盘够大 |
| **HD-1080p** | 下载 1080p | 画质够用、下载快、省空间 |
| **HD-720p** | 下载 720p | 只需要小文件 |
| **Any** | 什么画质都下 | 不挑画质 |

**推荐**：
- 看 4K 电视 → 选 **Ultra-HD**
- 普通电视/平板/手机 → 选 **HD-1080p**

### 4. 选择根目录（Root Folder）

确认 **Root Folder** 是 `/movies`。这对应 Samba 共享里的 `/share/Movies`。

### 5. 点击 "Add + Search"

在页面下方，有两个按钮：

- **Add Movie**：只添加电影，不搜索
- **Add + Search**：添加电影并立即搜索下载

推荐点 **Add + Search**。

---

## 三、等待自动下载

添加后，Radarr 会做这些事：

1. 通过 Jackett 搜索 The Pirate Bay 等索引器
2. 按你选的画质配置排序
3. 自动把最好的版本发送给 qBittorrent
4. qBittorrent 开始下载
5. 下载完成后，Radarr 自动重命名并移动到 `/share/Movies`

你可以打开 qBittorrent 看进度：

```
http://192.168.1.7:8085
```

账号：`admin` / 密码：`pi123456`

## 四、查看下载进度

### 在 Radarr 里看

Radarr 首页会列出所有电影：

- **绿色勾**：已下载完成
- **蓝色进度条**：正在下载
- **红色感叹号**：下载出错

点击电影封面，可以看到详细状态和预计完成时间。

### 在 qBittorrent 里看

qBittorrent 页面更直观：

- **Progress**：下载进度百分比
- **Down Speed**：当前下载速度
- **Seeds**：种子数（越多越快）
- **ETA**：预计剩余时间

## 五、手动选择特定版本

如果你不想用 Radarr 自动挑的版本，可以手动选。

### 1. 找到已添加的电影

在 Radarr 首页，点击你想下载的电影。

### 2. 点击 "Search"

在电影详情页，找到 **Search** 标签或按钮，点击 **Manual Search**（手动搜索）。

### 3. 选择版本

Radarr 会列出所有找到的资源，每行显示：

- 文件名
- 大小
- 种子数（Seeds）
- 画质（如 Bluray-1080p、Bluray-2160p）
- 是否被拒绝（Rejected）

**选择原则**：

- 种子数越多越好
- 画质满足你的需求
- 没有被 "Rejected"

### 4. 点击下载图标

在你想要的版本右边，点击 **下载图标**（通常是一个向下的箭头）。

Radarr 会把这个版本发送给 qBittorrent。

## 六、替换已下载的版本

比如你下了一个 1080p，后来想要 4K。

### 1. 进入电影详情页

在 Radarr 首页点击这部电影。

### 2. 点击 "Search" → "Manual Search"

### 3. 选择更好的版本

找到你想要的 4K 版本，点击下载图标。

### 4. Radarr 会自动替换

新的版本下载完成后，Radarr 会自动删除旧版本，保留新版本。

> 注意：如果新版本更差（比如从 4K 换回 1080p），Radarr 可能不会自动替换。你需要先删除旧文件，再手动搜索下载。

## 七、删除一部电影

### 只从 Radarr 删除，保留文件

1. 进入电影详情页
2. 点击 **Edit**
3. 拉到最下面，点 **Delete**
4. 不勾选 "Delete movie folder and files"

### 彻底删除电影和文件

1. 进入电影详情页
2. 点击 **Edit**
3. 拉到最下面，点 **Delete**
4. **勾选** "Delete movie folder and files"

## 八、如何排除不想下的小组

有些小组压缩很狠（比如 YIFY），画质一般。你可以让 Radarr 自动排除。

### 1. 创建 Custom Format

1. 打开 **Settings → Custom Formats**
2. 点击 **+** 号新增
3. Name 填：`YIFY`
4. 在 Conditions 里点 **+** → 选择 **Release Title**
5. 填入：`YIFY`
6. 点 **Save**

### 2. 给这个 Format 打负分

1. 打开 **Settings → Profiles**
2. 选择你常用的 profile（如 Ultra-HD 或 HD-1080p）
3. 在下面找到 **YIFY** 这一行
4. 把分数改成 `-10000`
5. 点 **Save**

以后只要文件名里带 `YIFY`，Radarr 就会自动跳过。

## 九、常见问题

### Q1：为什么搜索不到结果？

可能原因：

- 你输入的是中文名，试试英文名
- Jackett 代理断了，检查 V2Ray 是否在运行
- 这部电影在 The Pirate Bay 上确实没有资源
- 网络连接超时，等一会再试

### Q2：为什么找到了但不下载？

看看那一行有没有显示 **Rejected**。把鼠标放上去，会显示拒绝原因，常见：

- 种子数太少
- 画质不符合当前 Profile
- 文件格式被排除

### Q3：下载速度很慢怎么办？

- 查看 **Seeds** 数量，越少越慢
- 老电影/冷门电影通常慢
- 可以切到 1080p 版本，通常种子更多
- 检查路由器是否做了端口映射（端口 `6881`）

### Q4：下载完了在哪看？

成品在 Samba 共享的 `Movies` 文件夹：

```
\\192.168.1.7\SharedFiles\Movies
```

或者在树莓派上：

```
/share/Movies
```

### Q5：怎么改成优先 4K？

添加电影时，**Quality Profile** 选 **Ultra-HD**。

如果想改全局默认，去 **Settings → Profiles → Ultra-HD**，把 `Remux-2160p` 和 `Bluray-2160p` 排在最上面。
